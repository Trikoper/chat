### This project is supported only on linux.
# Chat

## Getting Started

```bash
run git clone git@github.com:cristichiu/chat.git
```


### Prerequisites for Debian/Ubuntu

- [GCC](https://gcc.gnu.org/)
```bash
sudo apt install gcc
sudo apt install libsll-dev
sudo apt install make
```


### Prerequisites for Fedora/RedHat

- [GCC](https://gcc.gnu.org/)
```bash
sudo dnf install gcc
sudo dnf install openssl-devel
sudo dnf install make
```

### Setup and Running the Application

1.  **Start server:**
    Open a terminal and run the following command from the project root directory:

    ```bash
    ./server.sh
    ```

2.  **Start client:**
    Open another terminal and run the following command from the project root directory:

    ```bash
    ./client.sh
    ```

The application should now be running.
